## Git merge

**Git merge** - это команда, которая используется для слияния (объединения) изменений из одной ветки в другую, создавая общий коммит.

```bash=
git merge .
```

Она позволяет переносить историю коммитов из одной ветки в текущую, что часто используется для включения завершенных функций или исправлений обратно в основную ветку (например, <u>**main**</u> или <u>**master**</u>).
