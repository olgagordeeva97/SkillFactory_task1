## Git add

**Git add *[файл]*** - добавляет файл в индекс.
Чтобы добавить все файлы (кроме игнорируемых), используйте команду: 

```bash=
git add .
```
Она сообщает Git, какие файлы и изменения вы хотите включить в следующий снимок <u>**(коммит)**</u>.
